TITLE: glob and download Example - JavaScript
DESCRIPTION: An example demonstrating the combined use of download and glob. It first downloads all files and then uses glob to return a Files representation of the entire directory.

SOURCE: https://github.com/vercel/vercel/blob/main/DEVELOPING_A_RUNTIME.md#_snippet_19

LANGUAGE: javascript
CODE:
```
const { glob, download } = require('@vercel/build-utils');

exports.build = ({ files, workPath }) => {
  await download(files, workPath);
  return glob('**', workPath);
};
```
