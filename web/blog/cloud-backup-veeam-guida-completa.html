2. Backup Repository

Storage locale per backup primari, ottimizzato per recovery rapido:

# Configurazione Repository con deduplicazione

  -Server $server `

3. Scale-Out Backup Repository (SOBR)

Architettura scalabile che combina storage locale e cloud:

# Configurazione SOBR con tiering cloud

  -ExtentRepository $localRepo, $cloudRepo `
  -Policy $tieringPolicy

# Policy di tiering automatico

  -CloudTierAfterDays 30 `
